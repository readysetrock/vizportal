Informations d'identification intégrées
Les éditeurs peuvent associer des informations de connexion à un classeur ou à une source de données. Les personnes qui accèdent au classeur ou à la source de données seront automatiquement authentifiées pour se connecter aux données.
Autoriser les éditeurs à intégrer les informations d'identification dans classeur ou une source de données

Informations d'identification enregistrées
Les utilisateurs peuvent enregistrer leur mot de passe pour pouvoir se connecter à des sources de données sans devoir s'authentifier.
Autoriser les utilisateurs à enregistrer les mots de passe pour les sources de données
Les utilisateurs peuvent enregistrer leurs jetons d'accès OAuth pour se connecter à des sources de données telles que Salesforce et Google BigQuery sans devoir s'authentifier.
Autoriser les utilisateurs à enregistrer les jetons d'accès OAuth pour les sources de données
