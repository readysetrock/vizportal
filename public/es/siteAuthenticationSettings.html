Clientes conectados
Los clientes de Tableau como, por ejemplo, aplicaciones móviles y clientes de sincronización, entre otros, pueden permanecer autenticados en este sitio después de que el usuario proporcione credenciales de inicio de sesión la primera vez. Las conexiones se establecen usando tokens seguros que identifican a cada usuario y cliente de forma única, sin guardar las credenciales del usuario.
Permitir los clientes conectados en este sitio de Tableau Online.

Inicio de sesión único con SAML - SAML está habilitado - SAML no está habilitado
Para habilitar el inicio de sesión único, debe tener un proveedor de identidad (IdP) que admita SAML y direcciones de correo electrónico para todos los usuarios que se correspondan con direcciones de correo del IdP.
