Hospedar varios sitios en Tableau Server
Cree sitios independientes para distintas organizaciones en una única instalación de Tableau Server.
Los libros de trabajo, los datos y las listas de usuarios de cada sitio están aislados de los de otros sitios. Como administrador del servidor, solo usted puede ver todos los sitios y realizar acciones como crear sitios y llevar a cabo cambios en todo el servidor.
