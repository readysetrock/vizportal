Credenciais inseridas
Os publicadores podem anexar credenciais a uma pasta de trabalho ou fonte de dados. As pessoas que acessam a pasta de trabalho ou a fonte de dados serão automaticamente autenticadas para se conectarem aos dados.
Permitir que os publicadores incluam as credenciais em uma pasta de trabalho ou fonte de dados
Os publicadores podem agendar as atualizações de extração de dados para suas pastas de trabalho e fontes de dados para manter as extrações atualizadas.
Permitir aos publicadores que agendem as atualizações de extração de dados
Credenciais salvas
Os usuários podem salvar suas senhas para se conectarem às fontes de dados sem precisar fazer a autenticação.
Permite que os usuários salvem as senhas para as fontes de dados
Os usuários podem salvar seus tokens de acesso OAuth para se conectarem às fontes de dados, como Salesforce e Google BigQuery, sem precisar fazer a autenticação.
Permite que os usuários salvem os tokens de acesso OAuth para as fontes de dados
